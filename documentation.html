Alerts

Alert prominently displays critical, time-sensitive warnings or directions across every page so that users see it whenever they visit the site. Alert keeps people informed of the status of the system and which may or may not require the user to respond. This includes errors, warnings, and general updates.

Buttons

The button is an essential element of any design. It's meant to look and behave as an interactive element of your page.The button is available in the different type - Primary, link, icon and floating buttons.

Example of Buttons

U! kit predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control.

Primary Button

To indicate that a button represents a primary action, add the btn__primary-container to the container of primary buttons and add classname alongside btn__normal btn-primary on each button.

Link Button

To indicate that a button represents a link action, add the btn__link-container to the container of link buttons and add classname alongside btn__link btn__link-secondary on each button.

Icon Button

To indicate that a button represents a icon action, add the btn__icon-container to the container of icon buttons and add classname alongside btn__icon on each button.

Floating Button

Images

Images can be responsive to fit the parent's width, and also can be customised to be round shaped.

Example of Responsive Image

Images in this library are made responsive with class name responsive on img tag . This applies width: 100%; and height: auto; and two more properties to the image so that it scales with the parent width.

Example of Round Image

Images in this library are made round with class name round on img tag . This applies width: 100%; and height: auto; to the image so that it scales with the parent width and border-radius:50% for make this round in shape.
